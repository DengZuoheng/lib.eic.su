借书须知

搜索

搜索可使用书名, 作者, ISBN作为关键词, 其中, 以13位数字作为关键词的话, 会自动尝试匹配ISBN, 如果没有匹配的馆藏书籍, 将返回"什么也找不到"的结果;以作者为关键词搜索时, 作者必须为全名, 即, 作者不支持模糊搜索;书名支持模糊搜索.

搜索结果将显示封面, 书名, 作者, ISBN, 馆藏情况等基本信息, 可通过每一结果项的书名(标题)点击进入详细信息页查看详细信息. 另外, 搜索结果提供"预约"按钮, 如果"剩余"数小于等于0, "预约"按钮将不可用.

预约流程

预约有助于提高借书成功率和简化借书流程, 强烈建议借书前预约

在"搜索结果", "书籍详情", "参看所有馆藏"几个页面中提供了相应的"预约"按钮, 点击进入预约页面.

进入预约页面后, 预约者填写学号, 姓名, 长短号(长号必填, 短号随便), 选择预约册数, 提交. 取书时需出示校园卡等身份证明, 所以请填写真实信息.

如果用户之前就预约过, 输入学号后, 会自动检索过往自动完成剩余各项.

如果该学号已经多次逾期归还, 损坏或遗失书籍, 将不能再预约(同样不能借书), 此时, 表单将警告且不能提交

提交后, 如果成功创建预约记录, 将跳转至预约成功页面, 此时, 将显示当前值班人员的联系方式, 请预约者务必记下联系方式并在24小时内联系值班人员安排取书取书事宜, 否则预约将被取消. 取书流程参考"借书流程(一般用户版)".

借书流程(一般用户版)

如果有提前预约, 值班人员会安排时间地点(应该是A307没跑了)取书, 取书时, 先报上学号姓名让值班人员找到预约记录, 然后在值班人员协助下到书架取下相关书籍. 值班人员记下当前书籍的破损情况(作为还书时是否损坏书籍的参考), 即可完成借书流程.

如果没有预约, 需在有值班人员值班的时间到A307办理借书, 借书时, 需提供学号, 姓名, 长号, 短号(可以不提供, 毕竟有些孩子没有短号).
